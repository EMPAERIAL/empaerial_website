# Context Glossary

## Landing Theme
The visual language used on the public landing-facing pages, including typography, spacing rhythm, surface treatment, and motion tone.

## Admin Theme Parity
Admin pages must use the same visual language as the Landing Theme, while keeping admin-specific information architecture and workflows.

## Admin Structure
The admin page layout and interaction model (tabs, editors, create/manage modes, CRUD flows). This remains admin-specific and is not replaced by landing page section order or marketing layout elements.
